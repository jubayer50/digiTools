## DigiTools

This is an amazing React project called DigiTools. Where someone can take a subscription for digital tools like AI and Content Writer, like this. This project has lots of functions, like the cart section, when someone adds to the cart, then how much producs has in the cart will show on the navbar cart icon, also the cart section tab. If someone has already added a product to the cart, one product after that want to add the same product, at that time give an alert that the same product is already in the cart. Another thing is that wehn some one click on paymet button, then all added products will be emptied and an empty message will show. This project is fully built using the JavaScript library React.js.

---

## The technologies used to create this project are: 
- React.js,
- TailwindCSS
- The most popular Tailwind component library DaisyUi.

---

## There are some features in this project.

- When an added product is in the cart, the number of items will show in the navbar on the cart icon.
- When someone clicks on the buy now button, then the tostify will show to ensure the product is successfully added to the cart same as remove item from the cart
- The most amazing feature is when you click on the process to checkout button, the cart section will empty, and an empty massage show in the cart section
